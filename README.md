```
Traceback (most recent call last):
  File ".\init.py", line 20, in <module>
    my_messages.backups() # 备份爬虫信息
NameError: name 'my_messages' is not defined
```

<p>
<p>
多半的原因是 BitVector 模块用不了，可自行调试。
